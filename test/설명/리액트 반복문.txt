 React의 반복문은 Map

  map의 사용법
    map을 사용시 태크안에 key를 넣어줘야함
    고유한 키를 넣어야함 > 중복되는 키가 존재하면 에러가 나옴
    화살표함수 = return 생략 가능 {중괄호} 제거해야함
    [ex].map((v, i) => {})처럼 선언이 가능 대신 i는 순서를 나타냄
    배열은 따로 밖으로 빼내도 괜찮음

    !!props란 컴포넌트에서 인자를 받아오기 위해 사용!

    <ul>
      {['1','2','3','4','5'].map( (v) => {
        return (
          <li>{v}</li>
        );
      })}
    </ul>
      실행시 배열이 리턴됨.
      반복되는 것을 맵 배열에 넣어주고 v로 불러와서 리턴받는다.

  다중 배열 - 1
    <ul>
      {[
        ['1', '1-1'],
        ['2','2-2'],
        ['3','3-3'],
        ['4','4-4],
        ['5','5-5]
      ].map( (v) => {
        return (
          <li>{v[0]} - {v[1]}</li>
        );
      })}
    </ul>
  
  다중 배열 - 2
    <ul>
      {[
        { first: '1', second: '1-1'}, 
        { first: '2', second: '2-2'}, 
        { first: '3', second: '3-3'}, 
        { first: '4', second: '4-4'}, 
        { first: '5', second: '5-5'}, 
      ].map( (v) => {
        return (
          <li key={v.first + v.second}>{v.first} - {v.second}</li>
        );
      })}
    </ul>
